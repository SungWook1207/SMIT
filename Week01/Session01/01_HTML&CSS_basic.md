# HTML CSS Basic

### 웹사이트의 기본 개념
- 웹사이트는 "클라이언트-서버 모델"을 따른다.
- 클라이언트(브라우저)에서 요청을 보내면 서버에서 응답을 보낸다.
- 프론트엔드(HTML, CSS, JavaScript)와 백엔드(Node.js, DB 등) 역할 이해.

### HTML & CSS 기초 실습
- HTML: 구조를 담당하는 언어
- CSS: 스타일을 적용하는 언어
